# miemhpc_challenge_26
MIEM HPC Challenge 2026 - MIEM/HSE

## A. КилоНОД

| Ограничение времени   | 10 секунд                         |
|-----------------------|:----------------------------------|
| Ограничение памяти    | 128 Мб                            |
| Ввод                  | стандартный ввод или input.txt    |
| Вывод                 | стандартный вывод или output.txt  |

Даны два целых числа a и b. Найдите их наибольший общий делитель (НОД).

### Условия
Числа помещаются в 64-х битный знаковый целый тип. Гарантируется, что оба числа не равны нулю одновременно.

### Формат ввода
В единственной строке через пробел заданы два целых числа a и b.

### Формат вывода
Выведите одно целое число gcd(a, b) — наибольший положительный общий делитель a и b.
